# 블로그 백엔드 서버
- 게시글 작성/수정/삭제, 댓글 작성/수정/삭제, 회원가입, 로그인 기능이 있는 블로그 백엔드 서버

# 사용 기술 및 개발 환경
- Java 17
- Spring Boot 2.7.2
- JPA
- Spring Security
- JWT
- Gradle
- IntelliJ

# API
https://documenter.getpostman.com/view/11069976/UzXNUH9U

## 1. 게시글 
- 게시글 작성
- 게시글 수정
- 전체 게시글 확인
- 개별 게시글 확인
- 개별 글 삭제 
  
## 2. 댓글
- 댓글 작성
- 댓글 수정
- 특정 게시글 전체 댓글 확인
- 특정 게시글 개별 댓글 확인
- 댓글 삭제

## 3. 유저
- 회원가입
- 로그인
- JWT 토큰
